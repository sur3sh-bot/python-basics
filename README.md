

# Python Basics Project

## Overview

This project demonstrates the fundamental concepts of Python programming. It is designed for beginners who want to understand the core building blocks of Python through simple examples and practical scripts.

The project covers basic programming constructs such as variables, data types, loops, functions, and conditional statements.

---

## Features

* Basic Python syntax examples
* Variables and data types
* Conditional statements (`if`, `elif`, `else`)
* Loops (`for`, `while`)
* Functions and reusable code
* Simple input and output handling
* Beginner-friendly structure


## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/python-basics-project.git
```

2. Navigate to the project directory

```bash
cd python-basics-project
```

3. Run the program

```bash
python main.py
```

---

## Example Code

```python
def greet(name):
    return f"Hello, {name}!"

user = input("Enter your name: ")
print(greet(user))
```

Output:

```
Enter your name: Alex
Hello, Alex!
```
